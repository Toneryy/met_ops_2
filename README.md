# Лабораторная работа: Поиск глобального экстремума одномерной липшицевой функции

## Описание

Программа реализует **метод Стронгина** для поиска глобального минимума одномерных липшицевых функций.

## Особенности

✅ **Метод Стронгина** - детерминированный алгоритм глобальной оптимизации  
✅ **Адаптивная оценка** константы Липшица  
✅ **Визуализация** с вспомогательной ломаной линией (оценка снизу)  
✅ **Сохранение результатов** в PDF (график + отчет)  
✅ **Тестовый пример** с функцией Растригина

## Входные параметры

В файле `main.py` можно настроить:

```python
func_str = "x**2 - 10*cos(2*pi*x) + 10"  # Функция
a, b = -5.12, 5.12                        # Границы отрезка
eps = 0.01                                 # Точность
```

## Выходные данные

Программа выводит:
- **x_min** - точка минимума
- **f_min** - значение функции в минимуме
- **Количество итераций**
- **Время работы**

И создает:
- **График** с функцией, точками испытаний и вспомогательной ломаной
- **PDF-файл** `result_strongin_method.pdf` с графиком и отчетом

## Примеры функций

### Функция Растригина (по умолчанию)
```python
func_str = "x**2 - 10*cos(2*pi*x) + 10"
a, b = -5.12, 5.12
eps = 0.01
```

### Функция Гриванка
```python
func_str = "x**2/4000 - cos(x) + 1"
a, b = -10, 10
eps = 0.01
```

### Функция с несколькими локальными минимумами
```python
func_str = "sin(x) + sin(10*x/3) + x**2/100"
a, b = -10, 10
eps = 0.01
```

## Алгоритм метода Стронгина

1. **Инициализация**: начинаем с границ отрезка [a, b]
2. **Оценка константы Липшица** M на основе текущих точек испытаний
3. **Применение параметра надежности**: m = r·M (r = 2)
4. **Вычисление характеристик** R(i) для всех интервалов
5. **Выбор интервала** с максимальной характеристикой
6. **Добавление новой точки** испытания
7. **Проверка критерия останова**: если длина максимального интервала < eps, то стоп

## Теория

**Липшицевая функция** - функция f(x), для которой существует константа L (константа Липшица), такая что:

```
|f(x₁) - f(x₂)| ≤ L·|x₁ - x₂|
```

**Метод Стронгина** гарантирует нахождение глобального минимума, используя:
- Адаптивную оценку константы Липшица
- Вспомогательную функцию (ломаную), которая является нижней оценкой целевой функции
- Детерминированную стратегию выбора точек испытаний
