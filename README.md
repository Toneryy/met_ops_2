# Поиск глобального минимума методом Стронгина

Лабораторная работа по реализации метода Стронгина для поиска глобального минимума одномерных функций.

## Что это такое?

Метод Стронгина — это алгоритм, который умеет находить самый низкий минимум функции, даже если у неё много локальных минимумов. В отличие от обычных методов оптимизации, которые могут "застрять" в локальном минимуме, метод Стронгина гарантированно находит глобальный минимум.

Алгоритм работает с липшицевыми функциями — это функции, которые не могут изменяться слишком быстро. Для таких функций можно построить нижнюю оценку и использовать её для поиска минимума.

## Что делает программа?

1. Принимает функцию и отрезок поиска
2. Находит глобальный минимум методом Стронгина
3. Строит график с функцией, точками испытаний и вспомогательной ломаной
4. Сохраняет результаты в PDF-файл

## Как запустить?

Сначала установите зависимости:

```bash
pip install -r requirements.txt
```

Затем запустите основную программу:

```bash
python main.py
```

Или запустите тесты на нескольких функциях:

```bash
python examples.py
```

## Как использовать?

В файле `main.py` можно изменить функцию и параметры поиска:

```python
func = rastrigin
a, b = -5.12, 5.12
eps = 0.01
```

Можно использовать свои функции или импортировать готовые из `examples.py`.

## Что получается на выходе?

Программа выводит в консоль:
- Точку минимума (x_min)
- Значение функции в минимуме (f_min)
- Количество итераций
- Время работы

И создаёт PDF-файл с графиком и отчётом.

## Как это работает?

Метод Стронгина работает так:

1. Начинаем с границ отрезка [a, b]
2. На каждой итерации оцениваем константу Липшица по уже вычисленным точкам
3. Для каждого интервала между соседними точками вычисляем характеристику R(i)
4. Выбираем интервал с максимальной характеристикой
5. Добавляем новую точку испытания в этот интервал
6. Повторяем, пока длина максимального интервала не станет меньше заданной точности

## Использование модуля в своих программах

Модуль `strongin.py` можно импортировать и использовать отдельно:

```python
from strongin import strongin_method

def my_function(x):
    return x**2 - 5*x + 6

x_min, f_min, iterations, time, points, values = strongin_method(
    my_function, a=0, b=5, eps=0.01
)
```

## Теория (кратко)

Функция называется липшицевой, если существует константа L такая, что:

|f(x₁) - f(x₂)| ≤ L·|x₁ - x₂|

Метод Стронгина использует это свойство для построения нижней оценки функции и гарантированного поиска глобального минимума.
